## 概要

+ コースニュースを年度・モジュールごとに月～金・１～６限の時間割形式で表示します。
+ 土日開講、長期休暇期間中開講、特定の時間が決められていない......など、時間割に配置できない一部コースニュースは時間割下部に列挙して表示します。
+ 表示されたコースニュースをクリックすることでそのコースニュースのページに飛ぶことができます。
+ 筑波大学のmanaba上においてのみ動作します。
